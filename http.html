HTTP Header Illustration

The HyperText Transfer Protocol (HTTP) is an application protocol used for transmitting hypermedia documents, such as HTML. It is the foundation of any data exchange on the Web and it is a protocol used for communication between web browsers and servers. HTTP follows a client-server model where the web browser, also known as the client, sends a request to the web server, which then sends back the requested information in the form of a response.

Request Line

Contains the method (e.g., GET, POST), the requested resource, and the HTTP version (e.g., HTTP/1.1).

Status Line

Contains the HTTP version, status code, and status message (e.g., HTTP/1.1 200 OK).

General Headers

Includes fields like Date, Connection, and Cache-Control.

Request Headers

Includes fields like Host, User-Agent, Accept, and Authorization.

Response Headers

Includes fields like Server, WWW-Authenticate, and Set-Cookie.

Entity Headers

Includes fields like Content-Type, Content-Length, and Content-Encoding.
