Source Port (16 bits)

Specifies the port number of the sending application.

Destination Port (16 bits)

Specifies the port number of the receiving application.

Sequence Number (32 bits)

Sequence number of the first byte in the segment.

Acknowledgment Number (32 bits)

If the ACK flag is set, this is the next sequence number expected.

Data Offset (4 bits)

Size of the TCP header in 32-bit words.

Reserved (3 bits)

Reserved for future use, set to zero.

Flags (9 bits)

NS (1 bit): ECN-nonce - Concealment protection
CWR (1 bit): Congestion Window Reduced
ECE (1 bit): ECN-Echo
URG (1 bit): Urgent pointer field significant
ACK (1 bit): Acknowledgment field significant
PSH (1 bit): Push function
RST (1 bit): Reset the connection
SYN (1 bit): Synchronize sequence numbers
FIN (1 bit): No more data from sender

Checksum (16 bits)

Error-checking for the header and data.

Urgent Pointer (16 bits)

Points to the sequence number of the byte following the urgent data.

Options and Padding (variable length)

Used for various options and to ensure the header length is a multiple of 32 bits.
