POP3 Header Illustration

POP3 Commands

USER (Maximum 40 characters)
PASS (Maximum 40 characters)
STAT (4 characters)
LIST (Variable length)
RETR (Variable length)
DELE (Variable length)
NOOP (4 characters)
RSET (4 characters)
QUIT (4 characters)

POP3 Responses

+OK (Maximum 512 characters)
-ERR (Maximum 512 characters)

USER (Maximum 40 characters)

Specifies the username for authentication. Back to header

PASS (Maximum 40 characters)

Specifies the password for authentication. Back to header

STAT (4 characters)

Requests the server to send a response with the number of messages and the total size of the mailbox. Back to header

LIST (Variable length)

Requests the server to send a list of messages with their sizes. Back to header

RETR (Variable length)

Requests the server to send the full content of a message. Back to header

DELE (Variable length)

Requests the server to delete a specific message. Back to header

NOOP (4 characters)

Sends a no-operation command to keep the connection alive. Back to header

RSET (4 characters)

Resets the session, marking all messages as not deleted. Back to header

QUIT (4 characters)

Terminates the session and deletes any messages marked for deletion. Back to header

+OK (Maximum 512 characters)

Indicates a positive response from the server. Back to header

-ERR (Maximum 512 characters)

Indicates a negative response from the server. Back to header
