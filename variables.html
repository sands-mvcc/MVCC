Introduction to Variables

In computer science, data is everything. But how do we store it? Meet the Variable.

1. What is a Variable?

Think of a variable as a labeled box in a warehouse.

  • The Box (Variable): A reserved space in the computer's memory.
  • The Label (Identifier): The name we give the box so we can find it later (e.g., score, username).
  • The Content (Value): The actual data stored inside (e.g., 95, "Alice").

Common Data Types

String

Text data. Always surrounded by quotes.

"Hello World"

Integer / Number

Whole numbers or decimals used for math.

42 or 3.14

Boolean

Logic data. Only two possible values.

true or false
