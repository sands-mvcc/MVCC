LDAP Header Illustration

LDAP (Lightweight Directory Access Protocol) is an open, vendor-neutral, industry-standard application protocol for accessing and maintaining distributed directory information services over an Internet Protocol (IP) network. LDAP is used to manage and access the distributed directory information services commonly used in network environments.

Message ID (Maximum 2147483647)

An integer that uniquely identifies a request in a session. Back to header

Protocol Operation (Variable length)

BER (Variable length)
