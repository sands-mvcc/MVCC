Complete Shortcuts List

Open New Chat
Ctrl + Shift + O
Start a fresh conversation without navigating through the interface. Perfect for beginning a new topic or clearing your current context.
Focus Chat Input
Shift + Esc
Jump directly to the chat input box from anywhere on the page. Ideal when your cursor is elsewhere and you want to start typing immediately.
Copy Last Code Block
Ctrl + Shift + ;
Instantly copy the most recent code block from ChatGPT's response to your clipboard. Great for developers and programmers.
Copy Last Response
Ctrl + Shift + C
Copy ChatGPT's most recent complete response to your clipboard. Useful for sharing or saving important answers.
Set Custom Instructions
Ctrl + Shift + I
Quickly access the custom instructions dialog to set your preferences for tone, style, and specific guidelines for ChatGPT to follow.
Toggle Sidebar
Ctrl + Shift + S
Show or hide the conversation sidebar to maximize your chat viewing area or access your conversation history.
Delete Chat
Ctrl + Shift + Backspace
Delete the current conversation. Use this when you want to remove a chat for privacy or organization purposes.
Show All Shortcuts
Ctrl + /
Display the complete list of available keyboard shortcuts directly within ChatGPT. Your quick reference guide.

Setup Guide

Follow these steps to start using ChatGPT keyboard shortcuts effectively.

Step 1: Access ChatGPT
Navigate to chat.openai.com and sign in to your account. The shortcuts work in any ChatGPT conversation.
Step 2: Open Shortcuts Help
Click the ? icon in the bottom-right corner of ChatGPT, then select "Keyboard shortcuts" to see the built-in shortcuts list.
Step 3: Practice Basic Shortcuts
Start with the most useful ones: Ctrl + / (help), Ctrl + Shift + O (new chat), and Shift + Esc (focus input).
Step 4: Customize Your Workflow
Use Ctrl + Shift + I to set custom instructions that align with your typical use cases and preferred response styles.
Step 5: Build Muscle Memory
Practice using shortcuts daily for a week. Focus on one or two shortcuts at a time until they become automatic.
