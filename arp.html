ARP Header Illustration

The Address Resolution Protocol (ARP) is a protocol used for mapping a network address (e.g., an IP address) to a physical address (e.g., a MAC address) that is recognized in the local network. ARP is a critical function in the networking stack, enabling devices to discover the hardware address of a device that is identified by its IP address.

Hardware Type (16 bits)

Specifies the type of hardware (e.g., Ethernet).

Protocol Type (16 bits)

Specifies the type of protocol (e.g., IPv4).

Hardware Address Length (8 bits)

Length of the hardware address (e.g., 6 for Ethernet).

Protocol Address Length (8 bits)

Length of the protocol address (e.g., 4 for IPv4).

Operation (16 bits)

Specifies the operation (e.g., request or reply).
