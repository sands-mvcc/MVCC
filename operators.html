Introduction to Conditions, Mathematical Operators, and Logical Logic

1. The Building Blocks of Logic

In computer science, programs make decisions using Conditions. A condition is a question that the computer evaluates to either true or false.

Mathematical Operators

Used to compare numbers.

OperatorMeaningExample
==Equal to5 == 5 (True)
!=Not Equal to5 != 3 (True)
>Greater than10 > 2 (True)
<Less than1 < 5 (True)
>=Greater/Equal5 >= 5 (True)

Logical Operators

Used to combine multiple conditions.

OperatorNameLogic
&&ANDTrue only if both sides are true.
||ORTrue if at least one side is true.
!NOTReverses the result (True becomes False).
