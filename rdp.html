RDP Header Illustration

Remote Desktop Protocol (RDP) is a proprietary protocol developed by Microsoft which provides a user with a graphical interface to connect to another computer over a network connection. The user employs RDP client software for this purpose, while the other computer must run RDP server software.

TS Header Type (8 bits)

Indicates the type of TPKT header. Back to header

X.224 Header Length (8 bits)

Specifies the length of the X.224 header. Back to header

X.224 Type (8 bits)

Indicates the type of X.224 message. Back to header

X.224 Class (8 bits)

Specifies the class of X.224 message. Back to header

X.224 Data Length (16 bits)

Indicates the length of the data segment in the X.224 header. Back to header

MCS Length (16 bits)

Specifies the length of the MCS PDU. Back to header

MCS PDU Type (32 bits)

Indicates the type of MCS PDU. Back to header

H.221 Nonstandard (32 bits)

Contains non-standard H.221 information. Back to header

Share Control Header (16 bits)

Specifies information for shared control data. Back to header

Share Data Header (16 bits)

Specifies information for shared data. Back to header
