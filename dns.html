DNS Header Illustration

The Domain Name System (DNS) is a hierarchical and decentralized naming system for computers, services, or other resources connected to the Internet or a private network. It translates domain names to IP addresses needed for locating and identifying computer services and devices with the underlying network protocols. DNS resolves queries for these names into IP addresses for the purpose of locating computer services and devices worldwide.

Identification (16 bits)

A 16-bit identifier assigned by the program that generates any kind of query. This identifier is copied in the corresponding response and can be used to match queries and responses. Back to header

QR (1 bit)

A one-bit field that specifies whether this message is a query (0) or a response (1). Back to header

Opcode (4 bits)

A four-bit field that specifies the kind of query in this message. This value is set by the originator of a query and copied into the response. The values are:

  • 0: Standard query (QUERY)
  • 1: Inverse query (IQUERY)
  • 2: Server status request (STATUS)
Back to header

AA (1 bit)

Authoritative Answer - this bit is valid in responses and specifies that the responding name server is an authority for the domain name in question section. Back to header

TC (1 bit)

TrunCation - specifies that this message was truncated due to length greater than that permitted on the transmission channel. Back to header

RD (1 bit)

Recursion Desired - this bit may be set in a query and is copied into the response. If RD is set, it directs the name server to pursue the query recursively. Recursive query support is optional. Back to header

RA (1 bit)

Recursion Available - this bit is set or cleared in a response and denotes whether recursive query support is available in the name server. Back to header

Z (3 bits)

Reserved for future use. Must be zero in all queries and responses. Back to header

RCode (4 bits)

Response code - this 4-bit field is set in responses and indicates the status of the response:

  • 0: No error
  • 1: Format error
  • 2: Server failure
  • 3: Name Error
  • 4: Not Implemented
  • 5: Refused
Back to header

QDCount (16 bits)

Specifies the number of entries in the question section. Back to header

ANCount (16 bits)

Specifies the number of resource records in the answer section. Back to header

NSCount (16 bits)

Specifies the number of name server resource records in the authority records section. Back to header

ARCount (16 bits)

Specifies the number of resource records in the additional records section. Back to header
