DNS Header Illustration

The Domain Name System (DNS) is a hierarchical and decentralized naming system used to translate human-readable domain names (e.g., www.example.com) into machine-readable IP addresses (e.g., 192.0.2.1). DNS is essential for the functionality of the internet, allowing users to access websites using domain names instead of numeric IP addresses. It operates through a network of DNS servers that communicate with each other to resolve domain names to IP addresses.

Identification (16 bits)

A unique identifier assigned by the program that generates any kind of DNS query.

Flags (16 bits)

Contains various control flags and status bits:

  • QR (1 bit): Query/Response Flag (0 for query, 1 for response)
  • Opcode (4 bits): Specifies the kind of query in the message (standard query, inverse query, or server status request)
  • AA (1 bit): Authoritative Answer Flag
  • TC (1 bit): Truncation Flag
  • RD (1 bit): Recursion Desired
  • RA (1 bit): Recursion Available
  • Z (3 bits): Reserved for future use (must be zero)
  • RCode (4 bits): Response Code (indicating errors or success)

Total Questions (16 bits)

Number of entries in the question section of the message.

Total Answer RRs (16 bits)

Number of resource records in the answer section.

Total Authority RRs (16 bits)

Number of resource records in the authority section.

Total Additional RRs (16 bits)

Number of resource records in the additional section.
