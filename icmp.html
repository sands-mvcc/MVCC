ICMP Header Illustration

Type (8 bits) Code (8 bits) Checksum (16 bits)
Rest of Header (32 bits)

Type (8 bits)

Indicates the type of ICMP message.

Code (8 bits)

Provides further information about the ICMP message type.

Checksum (16 bits)

Used for error-checking the ICMP header and data.

Rest of Header (32 bits)

This field varies based on the ICMP message type. It may include an identifier and sequence number for echo requests and replies, or a pointer to the error in the original message.
